# Exported Domain state is a mapping surface

Go Domain Entity fields may be exported to support explicit mechanical conversion among existing Application DTO, Domain Entity, and persistence Data Object state. Export does not authorize Application, Transport, or Infrastructure to implement business decisions from those fields or to mutate an existing Entity outside Domain behavior. `application/assembler.go` owns pure Application DTO/Domain Entity conversion, and `infrastructure/convert.go` owns pure Data Object/Domain Entity conversion; neither owns validation policy, I/O, logging, or transactions. Creating a new Aggregate is not an assembly operation and must call `domain.NewXxx` or another Domain-named Factory so valid initial state and creation events remain owned by Domain.
